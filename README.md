# java-filmorate

Backend-приложение для управления каталогом фильмов с социальной функциональностью. Проект позволяет пользователям оценивать фильмы, оставлять отзывы, добавлять друзей, получать рекомендации и отслеживать ленту событий. Разрабатывался командой из 5 человек с использованием feature-веток, pull request'ов и code review.

## 🛠️ **Стек технологий:** 
### Java 11+, Spring Boot, Spring JDBC, H2 Database, SQL, Jakarta Validation, JUnit 5, AssertJ, Lombok, Maven

## 🚀 Особенности проекта

- **Управление фильмами и пользователями** — полный CRUD для обеих сущностей
- **Лайки и рейтинги** — возможность ставить лайки фильмам, формирование топа популярных фильмов с фильтрацией по жанрам и годам
- **Система дружбы** — добавление в друзья, подтверждение, общие друзья
- **Отзывы** — пользователи могут оставлять и оценивать отзывы к фильмам
- **Лента событий** — отслеживание действий пользователей (лайки, отзывы, друзья)
- **Рекомендации** — подбор фильмов на основе предпочтений пользователя
- **Режиссёры** — привязка режиссёров к фильмам, поиск и фильтрация
- **Поиск** — поиск фильмов по названию и режиссёру с комбинированными параметрами
- **Общие фильмы** — получение фильмов, которые понравились двум пользователям

## 🏗️ Архитектура и технические решения

- Трёхслойная архитектура: controller → service → DAO
- Проектирование реляционной модели БД (many-to-many, внешние ключи, составные primary key)
- Работа с SQL через Spring JDBC (JOIN, GROUP BY, агрегатные функции)
- Валидация данных через Jakarta Bean Validation
- Unit-тестирование моделей и DAO-слоя
- Командная разработка с использованием Git (feature-ветки, code review)
